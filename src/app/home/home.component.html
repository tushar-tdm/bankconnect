Account Opening

IDBP Bank’s API for banking correspondence, facilitates account opening on behalf of the Bank. With services like Customer ID creation, savings account opening, duplication check etc., this API bundle offers necessary services to run a fully Digital Business Correspondence model. These APIs help in further enhancing and transforming the existing Bank offerings, by increasing their appeal to the existing and prospective customers alike..

UPI

The Unified Payments Interface (UPI) is an instant, real-time payment system developed by National Payments Corporation of India (NPCI) facilitating inter-bank transactions. It is a unique solution that can be used to transfer money and make payments using the Immediate Payments Service (IMPS)
